Download Detect Safe Browsing Software – It’s Free & Fast

Detect Safe Browsing (DSB) is an added layer of security we recommend all of our customers download on any device used to access online banking. If ever you do become victim to phishing (clicking from an email attempting to collect sensitive data), pharming (landing on a fake website that looks real) or malware (malicious software that infects your device), DSB will notify you of the infection and blocks the execution of malicious processes, allowing you to securely visit nbkc’s online banking and complete your activity.

DSB is a very fast & simple download, with no new passwords to remember and nothing to log in to. It runs quietly in the background and provides you with extra security when accessing your nbkc accounts online. Note: you should still maintain any malware or other fraud protection you use on your device – DSB is specific to protecting you while browsing on nbkc’s site to conduct banking activity.

How do I install Detect Safe Browsing?

In order to install the Detect Safe Browsing (DSB) client, follow the steps below:

For Microsoft Windows Platforms:

  • 1. Double click on the Detect Safe Browsing Installer.exe icon.
  • 2. A new window will display the download of DSB:
  • 3. Then, you can choose the installation language.
  • 4. The DSB installation wizard will open.
  • 5. Before starting the installation process, read carefully the license terms of agreement, check the box: “I have read and accept the terms…” and click on Next.
  • 6. Now, the installation of all DSB components will start.
  • 7. Once the installation is complete, you will see a confirmation message. Click on Finish.
  • Note: Once the installation is finished, restart your computer to ensure full protection.

For macOS Platforms:

  • 1. Double click on the Detect Safe Browsing (DSB) installer icon.
  • 2. The DSB installation wizard will open. Click on Continue.
  • 3. Before starting the installation process, read carefully the license terms of agreement. Click on Continue.
  • 4. Click on Agree to continue the installation.
  • 5. Now click on Install to start the installation of all DSB components.
  • 6. Once the installation is complete, you will see a confirmation message. Click on Close.
How do I uninstall Detect Safe Browsing?

For Microsoft Windows Platforms:

  • 1. Click on the Uninstall option located in the Detect Safe Browsing folder within the Windows Start Menu.

    DSB uninstalling process will require confirmation from the user through a CAPTCHA mechanism.
  • 2. A confirmation window will open. Click on Yes to continue.
  • 3. Detect Safe Browsing will be uninstalled.

For macOS Platforms:

  • 1. Find and run the Uninstall DSB file located in the same folder where the DSB application was initially installed from.
  • 2. After running the file, you will see an authentication window that corresponds to the operating system’s default confirmation message when making changes on the device.
  • 3. Once the uninstalling process has been confirmed, the DSB client will request additional confirmation on a different window. If you are sure about uninstalling the application, click on Ok, otherwise click on Cancel.
  • 4. Detect Safe Browsing will be uninstalled.
What are the system requirements for Detect Safe Browsing?

To install Detect Safe Browsing, your computer must meet the minimum system requirements below.

For Microsoft Windows Platforms:

  • Windows 7 SP1 (32-bit or 64-bit).
  • Windows 8 (32-bit or 64-bit).
  • Windows 8.1 (32-bit or 64-bit).
  • Windows 10 (32-bit or 64-bit).
  • Processor: Pentium 4 2.0 GHz or higher.
  • RAM Memory: 512 MB or higher.
  • Free hard disk space: 40 MB.
  • Internet connection: Wired, ADSL, or any equivalent broadband connection (512Kbps or higher).
  • You must have administrative permissions to install Detect Safe Browsing.

For macOS Platforms:

  • Operating System: macOS 10.10 or higher.
  • Processor: Intel Core 2 Duo.
  • System memory: 2 GB.
  • Free hard disk space: 20 MB.
  • Internet connection: Wired, ADSL, or any equivalent broadband connection (512Kbps or higher).
  • You must have administrative permissions to install Detect Safe Browsing.
